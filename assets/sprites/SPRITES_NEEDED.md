# Sprite Assets Needed

Add the following PNG sprite files to this folder before opening in Godot:

## Required Files

### `snow-tileset.png`
- The main snow/ice tileset for the TileMapLayers (Background, Playground, Foreground)
- Should be a tileset image with snow/ice/winter tiles
- Recommended: Use a free snow tileset from Kenney.nl or itch.io
- Suggestion: "Winter Platformer" or "Ice" tileset packs

### `slime_snow.png`
- Sprite sheet for the snow slime enemy (AnimatedSprite2D)
- Should have a "walk" animation (similar to slime_green.png from Main-Project)
- Frames: walk cycle (4-8 frames)

## Optional / Copy from Main-Project

You can copy these directly from Hanz-26/Main-Project (Hansel branch):
- `assets/sprites/Characters/` — player character sprites
- `assets/sprites/Hearts/` — heart UI sprites
- `assets/sprites/Items/` — coin/item sprites
- `assets/sprites/Weapons/` — sword/weapon sprites
- `assets/fonts/` — pixel font files

## Where to Find Free Assets

- https://kenney.nl/assets (free game assets, CC0 license)
- https://itch.io/game-assets/free (free category)
- Search: "snow platformer tileset", "winter 2D tileset", "ice platformer"
